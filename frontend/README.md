# 🎨 Fake News Analyzer - Frontend

Interface de usuário moderna e intuitiva para o sistema Fake News Analyzer, construída com **React** e **Vite**.

## 🛠️ Tecnologias

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Linguagem**: TypeScript
- **Roteamento**: [React Router 6](https://reactrouter.com/)
- **Estilização**: CSS Vanilla (Moderno e Responsivo)
- **Ícones**: [Lucide React](https://lucide.dev/) (ou similar)

## ✨ Funcionalidades do UI

- **Dashboard de Busca**: Interface limpa para inserção de URLs ou upload de arquivos.
- **Feedback em Tempo Real**: Estados de carregamento animados enquanto a IA processa a informação.
- **Visualização de Resultados**: Cards detalhados com código de cores para diferentes vereditos (Verde para Verdadeiro, Vermelho para Falso, etc.).
- **Página de Compartilhamento**: Visualização otimizada para links compartilhados.

## 🚀 Instalação e Execução

1. Instale as dependências:
   ```bash
   yarn install
   ```

2. Configure a URL da API (se necessário, geralmente aponta para localhost:3000 por padrão):
   ```env
   VITE_API_URL=http://localhost:3000
   ```

3. Inicie o ambiente de desenvolvimento:
   ```bash
   yarn dev
   ```

Interface focada em usabilidade e clareza visual para facilitar o consumo de informações verificadas.
